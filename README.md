# INGENZI HRMS

Human Resource Management System built with Tailwind CSS.

## Getting Started

Install dependencies:
```bash
npm install
```

Start development server:
```bash
npm start
```

Build for production:
```bash
npm run build
```

## Features

- Employee Management
- Department Management
- Leave Management
- Attendance Tracking
- Payroll Management
- Recruitment
- Reports & Analytics
- User Account Management
